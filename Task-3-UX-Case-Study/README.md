# Task 3 – UX Case Study
# Project Overview
This UX case study analyzes the user experience of the Zomato mobile application. The study evaluates the complete user journey, identifies strengths and weaknesses, and proposes practical design improvements to enhance usability and user satisfaction.

# Objective
- Analyze the overall user experience.
- Identify usability issues.
- Evaluate the application's strengths and weaknesses.
- Suggest practical UX improvements with redesigned mockups.

# Application Selected
**Zomato – Food Delivery Application**

## Report Includes
- Introduction
- Objective
- User Journey
- Strengths
- Weaknesses
- Three UX Improvements
- Current vs Improved Screens
- Conclusion

## Suggested UX Improvements
### 1. Simplify the Home Screen
- Reduced visual clutter
- Improved search visibility
- Better category organization

### 2. Improve Food Details Screen
- Added quick information card
- Better delivery information
- Improved content hierarchy

# 3. Simplify Checkout Process
- Added progress indicator
- Improved spacing and alignment
- Enhanced order summary
- Cleaner checkout experience

# Task 1 – Wireframing & Low-Fidelity Design

# Project Overview
This project focuses on creating low-fidelity wireframes for a Food Delivery mobile application. The wireframes were designed in Figma to establish the application's layout, navigation flow, and user experience before creating the final UI.

# Objective
- Design a simple and user-friendly mobile app layout.
- Plan the navigation flow between screens.
- Create a clear wireframe before designing the final interface.

# Wireframe Screens
- Splash Screen
- Login Screen
- Home Screen
- Food Details Screen
- Cart Screen
- Checkout Screen
- Order Success Screen

# Tools Used
- Figma
- FigJam

# Key Features
 Low-Fidelity Wireframes
- Simple User Flow
- Mobile-Friendly Layout
- Consistent Screen Structure
- Easy Navigation

# CodeAlpha UI/UX Design Internship

Welcome to my **CodeAlpha UI/UX Design Internship** repository.

This repository contains all the tasks completed during my UI/UX Design Internship at CodeAlpha. Each task demonstrates different aspects of the UI/UX design process, including wireframing, high-fidelity interface design, and user experience analysis.

# Repository Structure
📂 Task-1-Wireframing
📂 Task-2-High-Fidelity-UI
📂 Task-3-UX-Case-Study

# Completed Tasks

# Task 1 – Wireframing & Low-Fidelity Design
Created low-fidelity wireframes for a Food Delivery mobile application using Figma.

# Deliverables
- Splash Screen
- Login Screen
- Home Screen
- Food Details Screen
- Cart Screen
- Checkout Screen
- Order Success Screen

# Task 2 – High-Fidelity UI Design
Converted the wireframes into a modern high-fidelity UI using proper colors, typography, icons, and UI components.

# Features
- Modern Mobile UI
- Professional Color Palette
- Typography
- Icons & Images
- Interactive Prototype
- Consistent Design System
  
# Task 3 – UX Case Study
Conducted a UX case study of the Zomato application.

# Included
- Introduction
- Objective
- User Journey
- Strengths & Weaknesses
- Three UX Improvements
- Current vs Improved Mockups

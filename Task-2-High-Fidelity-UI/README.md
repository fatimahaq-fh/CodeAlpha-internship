# Task 2 – High-Fidelity UI Design
# Project Overview
This project transforms the low-fidelity wireframes into a modern, visually appealing, and user-friendly Food Delivery mobile application using Figma. The design follows UI/UX principles, ensuring consistency, accessibility, and an engaging user experience.

# Objective
- Convert wireframes into a polished UI.
- Apply color theory and typography.
- Use icons, images, and reusable UI components.
- Build a clean and consistent mobile interface
  
# Designed Screens
- Splash Screen
- Login Screen
- Home Screen
- Food Details Screen
- Cart Screen
- Checkout Screen
- Order Success Screen

# Design Elements
# Color Palette
- Primary Orange (#FF6B35)
- White (#FFFFFF)
- Light Gray (#F5F5F5)
- Dark Text (#222222)

# Typography
- Font Family: Poppins
- Clean and readable hierarchy
- Consistent spacing

# UI Components
- Buttons
- Cards
- Search Bar
- Bottom Navigation
- Icons
- Product Cards
  
# Tools Used
- Figma
- Iconify Plugin

# Features
- High-Fidelity UI
- Consistent Design System
- Interactive Prototype
- Modern Food Delivery Interface
- Mobile Responsive Layout
